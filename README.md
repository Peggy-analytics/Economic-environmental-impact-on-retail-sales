# Retail sales analysis: The impact of Economic and Environmental Factors
## Project review
This project analyzes how economic and environmental factors influence retail sales performance, The analysis was conducted using Excel dashboard,pivot tables, charts, and business intelligence techniques to uncover patterns, trends and actionable insights.
